# expo-slinky
Expo bindings for slinky (powered by ScalablyTyped)

It is distributed for Scala 2.13 and Scala.js 1

```
resolvers += Resolver.bintrayRepo("mcallisto", "libs")
libraryDependencies ++= Seq(
    "default" %%% "expo-slinky" % "0.1.1" 
) 
```
